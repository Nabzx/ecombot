# Cancellations Policy

Orders can be cancelled free of charge **while they have not yet been dispatched**.

- Cancellation is available while an order is in the placed, paid, or processing state
  and the shipment has not left the warehouse.
- Once an order has shipped, it can no longer be cancelled; the customer should instead
  refuse delivery or start a return once the item arrives.
- A cancelled order is refunded in full to the original payment method.

Requests to cancel a shipped order are redirected to the returns process.
